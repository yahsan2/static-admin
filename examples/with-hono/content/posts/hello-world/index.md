# Welcome to Static Admin

This is a sample blog post created with Static Admin, a Git-based CMS inspired by Keystatic.

## Features

- **File-based content**: All content is stored as Markdown files
- **Git integration**: Auto-commit on save
- **TipTap WYSIWYG**: Rich text editing
- **SQLite auth**: Simple authentication

## Getting Started

1. Run `pnpm setup` to create an admin user
2. Visit `/admin` to access the admin panel
3. Start creating content!

Happy writing!
